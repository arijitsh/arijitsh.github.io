References
==========

ttc builds on a line of work on model counting for SMT by Arijit Shaw,
Kuldeep S. Meel, and collaborators. Each of the four engines is described in
one of the following papers.

- :cite:`SSM25` — the LRA volume-computation engine (:doc:`../volume/index`).
- :cite:`SM25` — projection-based approximate counting for hybrid formulas
  (:doc:`../hybrid/index`).
- :cite:`SJM24` — Skolem-function counting (:doc:`../function/index`).
- :cite:`SM24` — counting and sampling for bit-vectors
  (:doc:`../bitvectors/index`).
