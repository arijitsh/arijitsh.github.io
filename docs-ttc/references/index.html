References

ttc builds on a line of work on model counting for SMT by Arijit Shaw, Kuldeep S. Meel, and collaborators. Each of the four engines is described in one of the following papers.

  • [SSM25] — the LRA volume-computation engine (Volume Computation).

  • [SM25] — projection-based approximate counting for hybrid formulas (Hybrid Formulas).

  • [SJM24] — Skolem-function counting (Function Counting).

  • [SM24] — counting and sampling for bit-vectors (Counting on Bitvectors).
